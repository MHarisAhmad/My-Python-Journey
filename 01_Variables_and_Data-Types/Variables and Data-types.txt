Concept: Variables & Data Types
In Python, a variable is like a labeled container or a storage box. It holds data that your program can use and change later.

Unlike other programming languages (like Java or C++), Python is dynamically typed. This means you don't need to declare what kind of data goes into the box ahead of time. Python figures it out automatically based on what you put inside.

The 4 Basic Data Types You Must Know:
1. Integer (int): Whole numbers, positive or negative. (e.g., age = 17)

2. Float (float): Numbers with decimals. (e.g., percentage = 94.5)

3. String (str): Text wrapped in single or double quotes. (e.g., name = "Rahul")

4. Boolean (bool): True or False logical values. (Note the capital T and F). (e.g., is_passed = True)

Rules:
1. Naming Rules: Variable names can only start with a letter or an underscore (_). They cannot start with a number, and they cannot contain spaces or special characters (like @, $, %).

2. Case Sensitivity: myAge, myage, and MYAGE are three completely different variables.

3. No Reserved Words: You cannot use Python's built-in keywords (like if, else, print, True) as variable names.
